================================================================================
                        COMO COMEÇAR
================================================================================

1. EXTRAIR PROJETO
   tar -xzf yms-axioma-android-complete.tar.gz
   cd yms-axioma-app

2. INSTALAR DEPENDÊNCIAS
   npm install --legacy-peer-deps

3. COMPILAR PROJETO WEB
   npm run build

4. SINCRONIZAR COM ANDROID
   npx cap sync android

5. ABRIR NO ANDROID STUDIO
   open android/
   (ou: start android\ no Windows)

6. COMPILAR E EXECUTAR
   - Conectar dispositivo ou iniciar emulador
   - Clicar em Run (▶️) no Android Studio

================================================================================
                        PRÓXIMOS PASSOS
================================================================================

PARA TESTES LOCAIS:
   cd android
   ./gradlew assembleDebug
   adb install app/build/outputs/apk/debug/app-debug.apk

PARA PUBLICAÇÃO NA PLAY STORE:
   1. Criar certificado de assinatura
   2. Compilar APK/AAB release
   3. Criar conta Google Play Developer
   4. Seguir guia em ANDROID_BUILD_GUIDE.md
   5. Fazer upload na Play Store

IMPLEMENTAR MODO MOTORISTA (OPCIONAL):
   - Seguir instruções em DRIVER_MODE_CONFIG.md
   - Modificar src/App.tsx e src/pages/RoleSelection.tsx
   - Recompilar com npm run build
